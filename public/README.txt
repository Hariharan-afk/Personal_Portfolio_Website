Public Assets Folder
====================

Place your assets in this folder:

1. resume.pdf - Your resume/CV file (required for the download button to work)
2. images/ - Create this folder for project screenshots, logos, etc.
3. favicon.ico - Your website favicon (optional)
4. Any other static files you want to serve

The resume.pdf file is referenced in the Hero section's "Download Resume" button.
Make sure to add your actual resume here before deploying!
